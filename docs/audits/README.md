# Coverage Audits

This folder holds coverage audit reports for Phylax.

## Convention

- `current-coverage.md` is the canonical, latest audit. There is only ever one.
- Before writing a new audit, the previous `current-coverage.md` is archived to `history/YYYY-MM-DD-coverage.md`, where the date is the audit date of the OLD report.
- The `history/` folder contains frozen snapshots for comparison over time. Files there are never edited after archiving.

## Date format

File names in `history/` use ISO 8601 date format: `YYYY-MM-DD-coverage.md`.
